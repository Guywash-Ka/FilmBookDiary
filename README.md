# Fibo
This Android app is used for keeping notes of films and books. This is a diary, where you can add notes of any films and books, review and rate them.


# Installation
* Download APK-file on your phone

    https://github.com/Guywash-Ka/FilmBookDiary/blob/master/app-debug.apk
### Or

* Run in Android Studio
  1. Clone app to your project
  2. Run project via Run 'app'
  

# Technical side

* Jetpack Compose
* Room database
* MVVM architecture
* Navigation Compose

# Requirements

* Minimal SDK version - 28
* Target SDK - 33
* Kotlin version - 1.7.0
